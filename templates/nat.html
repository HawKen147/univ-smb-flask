Les types de NAT Principe
NAT Statique

Pour un accès permanent à une ressource sur un réseau privé depuis Internet. Utilisé pour un serveur web s'il est contraint d'être placé sur un réseau privé. Annule le bénéfice des IP privées car ne diminue pas le besoin en IP publiques (une IP privée correspond une IP publique).
NAT dynamique avec surcharge = NAT overload = PAT

C'est celui utilisé par les Box Internet grand public.N IP privées correspond une seule IP publique
